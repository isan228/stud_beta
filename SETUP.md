# Инструкция по установке и запуску

## Требования

- Node.js (версия 14 или выше)
- PostgreSQL (версия 12 или выше)
- npm или yarn

## Шаги установки

### 1. Установка зависимостей

```bash
npm install
```

### 2. Настройка базы данных

Создайте базу данных PostgreSQL:

```sql
CREATE DATABASE stud_kg;
```

### 3. Настройка переменных окружения

Создайте файл `.env` в корне проекта (скопируйте из `env.example`):

```bash
cp env.example .env
```

Отредактируйте `.env` и укажите свои данные:

```
PORT=3000
DB_HOST=localhost
DB_PORT=5432
DB_NAME=stud_kg
DB_USER=postgres
DB_PASSWORD=ваш_пароль
JWT_SECRET=ваш_секретный_ключ_для_jwt
NODE_ENV=development
```

### 4. Инициализация базы данных

Запустите скрипт для создания таблиц и тестовых данных:

```bash
npm run init-db
```

Это создаст:
- Все необходимые таблицы
- Тестовые предметы (Математика, Физика, Химия, История, Биология)
- Тестовые тесты с вопросами и ответами

### 5. Запуск сервера

Для разработки (с автоперезагрузкой):
```bash
npm run dev
```

Для продакшена:
```bash
npm start
```

### 6. Открытие в браузере

Откройте браузер и перейдите по адресу:
```
http://localhost:3000
```

## Первое использование

1. Зарегистрируйтесь, создав новый аккаунт
2. Выберите предмет из списка
3. Выберите тест
4. Настройте параметры теста (количество вопросов, таймер, случайный порядок)
5. Начните прохождение теста
6. Добавляйте понравившиеся вопросы в избранное
7. Просматривайте статистику в личном кабинете

## Возможные проблемы

### Ошибка подключения к БД
- Убедитесь, что PostgreSQL запущен
- Проверьте правильность данных в `.env`
- Убедитесь, что база данных `stud_kg` создана

### Порт занят
- Измените `PORT` в `.env` на другой порт
- Или остановите процесс, использующий порт 3000

### Ошибки при инициализации БД
- Убедитесь, что пользователь БД имеет права на создание таблиц
- Проверьте, что база данных существует
