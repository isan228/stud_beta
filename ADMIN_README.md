# Инструкция по входу в админ-панель

## Создание первого администратора

### Способ 1: Через скрипт (рекомендуется)

1. Откройте терминал в корне проекта
2. Выполните команду:
```bash
npm run create-admin
```

Это создаст администратора с данными по умолчанию:
- **Имя пользователя:** admin
- **Email:** admin@stud.kg
- **Пароль:** admin123

### Способ 2: С указанием своих данных

```bash
npm run create-admin username email password
```

Например:
```bash
npm run create-admin myadmin admin@example.com mypassword123
```

### Способ 3: Через Node.js напрямую

```bash
node scripts/createAdmin.js username email password
```

## Вход в админ-панель

1. Откройте браузер
2. Перейдите по адресу: **http://localhost:3000/admin**
3. Введите данные администратора:
   - Имя пользователя (username)
   - Пароль
4. Нажмите "Войти"

## Функционал админ-панели

После входа вы получите доступ к:

- **Дашборд** - общая статистика платформы
- **Пользователи** - управление пользователями (просмотр, поиск, удаление)
- **Предметы** - управление предметами (создание, редактирование, удаление)
- **Тесты** - управление тестами (создание, редактирование, удаление)
- **Вопросы** - управление вопросами и ответами (создание, редактирование, удаление)

## Безопасность

⚠️ **Важно:** После первого входа рекомендуется изменить пароль администратора через базу данных или создать нового администратора с более надежным паролем.
