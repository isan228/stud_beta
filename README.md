# stud.kg - Платформа для подготовки к экзаменам

Веб-приложение для студентов, позволяющее готовиться к экзаменам через систему тестирования.

## Функционал

- **Регистрация и авторизация** - безопасная система аутентификации
- **Тесты по предметам** - множество тестов по различным предметам
- **Настройка тестов**:
  - Выбор количества вопросов
  - Случайный порядок ответов
  - Таймер для прохождения теста
- **Избранное** - сохранение вопросов для повторения
- **Тест из избранного** - прохождение теста только из избранных вопросов
- **Статистика** - отслеживание прогресса, стрики (серии дней)
- **Темная/светлая тема** - переключение между темами
- **Современный дизайн** - анимации и плавные переходы

## Технологии

- **Backend**: Node.js, Express.js
- **База данных**: PostgreSQL
- **ORM**: Sequelize
- **Frontend**: HTML, CSS, JavaScript (Vanilla)
- **Аутентификация**: JWT

## Установка

1. Установите зависимости:
```bash
npm install
```

2. Создайте файл `.env` на основе `.env.example`:
```
PORT=3000
DB_HOST=localhost
DB_PORT=5432
DB_NAME=stud_kg
DB_USER=postgres
DB_PASSWORD=your_password
JWT_SECRET=your_jwt_secret_key_here
NODE_ENV=development
```

3. Убедитесь, что PostgreSQL запущен и создана база данных `stud_kg`

4. Инициализируйте базу данных (создаст таблицы и тестовые данные):
```bash
npm run init-db
```

5. Запустите сервер:
```bash
npm start
```

Для разработки с автоперезагрузкой:
```bash
npm run dev
```

6. Откройте браузер и перейдите на `http://localhost:3000`

## API Endpoints

### Аутентификация
- `POST /api/auth/register` - Регистрация
- `POST /api/auth/login` - Вход
- `GET /api/auth/me` - Получить текущего пользователя

### Тесты
- `GET /api/tests/subjects` - Получить все предметы
- `GET /api/tests/subjects/:subjectId/tests` - Получить тесты по предмету
- `GET /api/tests/tests/:testId` - Получить конкретный тест
- `POST /api/tests/tests/:testId/questions` - Получить вопросы с настройками
- `POST /api/tests/tests/:testId/check` - Проверить ответы

### Избранное
- `GET /api/favorites` - Получить все избранные вопросы
- `POST /api/questions/:questionId/favorite` - Добавить в избранное
- `DELETE /api/questions/:questionId/favorite` - Удалить из избранного
- `GET /api/questions/:questionId/favorite` - Проверить статус избранного

### Статистика
- `GET /api/stats/stats` - Получить статистику пользователя
- `POST /api/stats/test-result` - Сохранить результат теста

## Использование

1. **Регистрация**: Создайте аккаунт, указав никнейм, email и пароль
2. **Выбор предмета**: Выберите предмет из списка
3. **Выбор теста**: Выберите тест по предмету
4. **Настройка**: Настройте параметры теста (количество вопросов, таймер, случайный порядок)
5. **Прохождение**: Отвечайте на вопросы, добавляйте понравившиеся в избранное
6. **Результаты**: Просматривайте результаты и статистику
7. **Избранное**: Повторяйте избранные вопросы или проходите тест из них
